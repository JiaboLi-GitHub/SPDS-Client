# SPDS_Client

**坐姿检测系统客户端**

## 项目介绍

​         本项目能够检测用户坐姿并给予反馈，项目包含客户端和服务端，实现了登陆、注册、坐姿检测、数据可视化、关联账户数据浏览等功能。  

服务端：https://github.com/JiaboLi-GitHub/SPDS-Server

## 项目开放所用技术

- C++

- [Qt](https://www.qt.io/zh-cn/)

  

## 实现功能

1. 使用Qt提供的QTcpSocket类进行套接字通讯。
2. 多线程
3. 数据库sqlite
4. 可视化
